# A Markdown application

This is a simple application that renders markdown to the top half of the screen.

It is using React, Typescript, and vitest for unit testing.

**To start:**

`npm i`

`npm run dev`

## Features for future Development

- Open
- Save
- Undo
- Redo
- Inline editing
